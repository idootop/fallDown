# 简介 

FallDown is a game developed using Flutter+Flame+Box2D.

坠落是使用[Flutter](https://flutter.dev/)+[Flame](https://flame-engine.org/)+[Box2D](https://flame-engine.org/docs/box2d.md)开发的一个开源小游戏。
